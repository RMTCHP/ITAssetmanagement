Snipe-IT Google Sheets Integration

Welcome to the Snipe-IT project! This project is a customized version of Snipe-IT that integrates with Google Sheets for asset management. It allows you to manage your organization's assets efficiently using a simple and familiar interface.

Project Description

Snipe-IT is an open-source asset management system designed to track and manage your organization's assets, such as computers, phones, and other equipment. This version of Snipe-IT uses Google Sheets as the data backend, making it easy to manage and share asset data across your organization.

Getting Started

  1. Clone the project from GitHub: 
    git clone https://github.com/username/repository-name.git
  2. Install dependencies using Composer: 
    composer install
  3. Configure the Google Sheets API:
    1. Enable the Google Sheets API in your Google Cloud Console.
    2. Create credentials (API key or OAuth2 credentials) for your application.
    3. Update the .env file with the necessary credentials.
  4. Run the application and start managing your assets through the web interface.

Project Structure

The project uses multiple Google Sheets to store different types of data. Below is an overview of the structure:

  • Assets: Stores information about assets including asset ID, name, category, purchase date, price, assigned user, and status.
  • Users: Contains user information such as user ID, name, email, department, and role.
  • AssetAssignments: Tracks asset assignments, linking assets to users with assignment and return dates.
  • SoftwareLicenses: Manages software licenses, including license ID, software name, license key, assigned user, and expiration date.
  • AssetCategories: Defines categories for assets, each with a unique ID and name.
  • AssetMaintenance: Logs maintenance activities for assets, including maintenance ID, asset ID, date, type, cost, and technician.
  • UsageHistory: Records the usage history of assets, tracking usage periods and associated users.
  • Vendors: Lists vendors, including vendor ID, name, contact information, and address.
